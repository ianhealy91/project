# Logbook
 
A web-based job application tracker built with ASP.NET Core MVC and C#, developed as a capstone project for the BSc Honours in Contemporary Software Development programme at Atlantic Technological University (ATU) Donegal.
 
Logbook enables jobseekers to log, manage, and report on their job search activity. It is designed with particular consideration for recipients of Ireland's Jobseeker's Allowance who are required to evidence active job-seeking to the Department of Social Protection (DSP).
